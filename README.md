# langchain-demo

This is a demo project to show case how LangChain agents can pull information across multiple systems e.g. Linkedin
Twitter etc. and to summerize information about user using a large language model (LLM) in our case OpenAI. 

The ice_breaker.py will run and provide the info in the about any user who has public profile in twitter and linkedin.
